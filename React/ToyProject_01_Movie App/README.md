# Movie_App
React.js로 만든 Movie App 토이 프로젝트 (노마드 코더 클론 코딩)



## 기능

- MovieList
- MovieDetail
- Navigation

## 스택

- React.js
- CSS

## 설명

- yts API (axios)
- HashRouter
- gh-pages 배포
- Context API를 활용하여 State관리 (연습)
