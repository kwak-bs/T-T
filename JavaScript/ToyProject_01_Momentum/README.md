# Momentum
노마드 코더 강좌중 하나인 Momonton 클론 코딩 토이 프로젝트입니다. 

## 기능
- Clock
- To Do List
- Weather
- Location
- Daily Background Photo

## 스택
- Vanila JS
- HTML5
- CSS

## 설명

- Unsplash API 사용 (배경사진)
- OpenWeather API 사용 (날씨)
- LocalStorage 활용
- gh-pages 배포
- navigator.geolocation.getCurrentPosition()  // 위치 정보 권한 허용
