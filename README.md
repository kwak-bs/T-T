# TIL - ToyProject
#### 각 언어별로 공부한 내용과 혼자 진행했던 ToyProject 모음집입니다. 



## 설명
- 공부한 내용을 까먹지 않기 위해 저장하고 있습니다. 
- 진행한 ToyProject들을 언어 별로 정리하고 저장하고 있습니다. 
- 꾸준히, 묵묵히 진행하는 것이 목표입니다. 
- 반복적으로 자주 보기위해 짧게 짧게 내용을 작성한건 비밀입니다.
